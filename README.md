# fiskaly SMAERS

Note, that this Security Module Application for Electronic Record-keeping Systems (SMAERS) component is **not certified** (yet)!

## Available Platforms: 

- android/386 (x86)
- android/arm
- android/arm64

- darwin/386 (x86, macOS)
- darwin/amd64 (macOS)
- darwin/arm (iOS)
- darwin/arm64 (iOS)

- js/wasm (Web Browser)

- linux/386 (x86)
- linux/amd64
- linux/arm
- linux/arm64

- windows/386 (x86)
- windows/amd64
